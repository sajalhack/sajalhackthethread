## Quick Start

1. **Start your Flask app:**
   ```bash
   python app.py
   ```

2. **Start ngrok** (in another terminal):
   ```bash
   ngrok http 5002
   ```

3. **Configure Twilio webhook** with your ngrok URL

4. **Send a link via WhatsApp** to your bot number

5. **View on dashboard:**
   - Open `http://localhost:5002`
   - See all your saves
   - Search by category or content
   - Use "Random Inspiration" button
